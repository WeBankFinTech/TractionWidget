# BNavBar
提供页面上和抽屉内的导航栏

## 组件注册

```js
import { BNavBar } from '@webank/bdp-design';
app.use(BNavBar);
```

## 参数说明
### BNavBar Props
| 属性  | 说明                   | 类型                                    |  默认值                                 |是否必须|
| ----- | ----------------------------- | ---------------------------------------- |------------------ |----- |
| curPath | 当前路径 | String | '' | 是
| data | 导航栏数据 | Array | [] | 是
| layout | 导航栏模式，分为page-页面上和drawer-抽屉内两种 | String | page | 否
### BNavBar Events
| 事件名称          | 说明                                                                                                                                            | 回调参数                                |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------ |
| navChange | 导航栏切换时触发的回调函数                                                                                                                                          | 导航栏当前路径
