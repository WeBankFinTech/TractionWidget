# BNavHeader
提供页面顶部的标题返回栏

## 组件注册

```js
import { BNavHeader } from '@fesjs/traction-widget';
app.use(BNavHeader);
```

## 参数说明
### BNavHeader Props
| 属性  | 说明                   | 类型                                    |  默认值                                 |是否必须|
| ----- | ----------------------------- | ---------------------------------------- |------------------ |----- |
| title | 顶部栏标题 | String | - | 是
| isShowArrow | 是否显示返回箭头 | Boolean | true | 否
### BNavHeader Events
| 事件名称          | 说明                                                                                                                                            | 回调参数                                |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------ |
| back | 点击返回箭头时触发的回调函数                                                                                                                                          | -
### BNavHeader Slots
| 名称          | 说明                                                                                                                                            | 参数                                |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------ |
| suffix | 后缀内容                                                                                                                                          | -                        |
